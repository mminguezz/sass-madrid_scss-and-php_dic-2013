Importando, formatos de salida y comentarios

Ahora visitamos: example.com/index.php y nos creara el archivo css apartir de nuestro scss

Ejemplo: scss-b/index.php 


        # Creamos en archivo index.php
 <?php
        $directory = __DIR__."/scss";
        require "scss.inc.php";
        $scss = new scssc();
        $handle = fopen('style.css', 'w');
        $scssCode = $scss->compile(file_get_contents($directory.'/style.scss'));
        fwrite($handle, $scssCode);
        fclose($handle);
?>
